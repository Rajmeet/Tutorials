Bootstrap - Some tips

Bootstrap has a powerful container and grid based layout system that allows mobile first/ mobile friendly design as the web page reorganizes itself automatically with the changing screen sizes. This page viewed on a smaller screen (change your browser window size to test it out) will display differently with the two columns below stacked, instead of side by side...And this row,including the headline displays properly with appropriate padding no matter how much you shrink the page.
In fact the whole page below the navbar is inside what we call a container..I've used horizontal lines(<hr>) to demarcate each row.
Learn more here : Bootstrap Layout
